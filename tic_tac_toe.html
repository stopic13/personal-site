MicroPython Tic-Tac-Toe

This project was a fun way to learn more about MicroPython and get comfortable using a PyBoard! The program is ~200 lines of code and is run on a PyBoard with an LCD160CR display attached.

Features

  • Selects a player (X or O) at random to go first
  • Touch-sensitive LCD screen interprets presses and places pieces accordingly
  • Screen animates when someone gets three in a row or if the game ends in a tie
  • Press USR button at any time to start a new game!

cookieTracker

During my sophomore year at Dartmouth I did research on browser security and privacy. Part of my work included writing an add-on SDK extension for Firefox to display information about the cookies set in a typical browsing session. I presented this work at the 2016 Karen E. Wetterhahn Science Symposium.

Note: The technology use to develop this add-on is now considered legacy technology and is no longer supported by Firefox. Nevertheless, the code can be found here: and may be migrated to Web Extensions in the future.

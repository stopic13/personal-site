Second-Order Markov Sentence Generator

This web app generates sentences using a second-order Markov Chain. Users can select from a combination of pre-selected novels, or by default the app will pick one of the books at random to use as the state space. Try it out here!

Tech Stack

Frontend: React

Backend: Flask, Python 3.6
